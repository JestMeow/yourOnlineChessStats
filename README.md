# Overview
A web app to visualize your online chess data. Currently, only supports data for users from chess.com.

# Features
Player information
- Shows general information about the player, e.g., username and country.

Records
- Shows total wins, draws, and losses and their overall percentage.
- Current rating, peak rating, and rating deviation.

Rating
- Rating against time graph.

Openings
- Opening names
- Total games played
- Win/Draw/Loss
